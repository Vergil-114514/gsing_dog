# 项目说明

## 项目概述

四足机器狗（带机械臂与气泵）上位机 ROS2 工作空间。下位机已完成四腿运动学/步态控制、机械臂动作执行、气泵开关，上下位机通过 USB CDC 虚拟串口全双工通信。

**上位机职责**：激光雷达导航（Nav2）、视觉识别（目标检测/定位）、高层路径规划与策略执行（状态机/行为树）。

## 技术栈

- **ROS2**: Humble / Iron / Jazzy
- **C++17**: 核心控制节点（quadruped_control）
- **Python 3**: 视觉节点 + Launch 脚本
- **串口通信**: Linux termios（serial_bridge_node），Windows 下为 stub
- **构建工具**: colcon (--symlink-install)
- **Astra 相机**: openni2 驱动 + ROS2 封装（astra_camera）
- **3D 目标检测**: YOLO + 点云定位（detection_3d）
- **待集成**: Nav2, BehaviorTree.CPP

## 运行方式

### 环境前置条件

```bash
# Ubuntu 22.04 / 24.04 — 安装 ROS2 Humble (或 Iron/Jazzy)
sudo apt install ros-humble-desktop python3-colcon-common-extensions
source /opt/ros/humble/setup.bash
```

### 编译

```bash
cd d:/gsing_dog   # 或实际路径
colcon build --symlink-install
source install/setup.bash
```

> **注意**：serial_bridge_node 的串口 I/O 仅在 Linux 下可用（termios），Windows 下编译通过但串口操作为无操作 stub。

### 启动

```bash
# 最小启动（串口桥 + 策略节点）
ros2 launch quadruped_bringup robot.launch.py port:=/dev/ttyACM0 baudrate:=115200

# 加视觉
ros2 launch quadruped_bringup robot.launch.py use_vision:=true

# 加 LiDAR + Nav2（硬件就绪后）
ros2 launch quadruped_bringup robot.launch.py use_lidar:=true use_nav2:=true
```

### Topic 拓扑

```
strategy_manager ──LegCommand/ArmPumpCommand──▶ serial_bridge ──(USB CDC)──▶ MCU
strategy_manager ◀──RobotState────────────────── serial_bridge ◀──(USB CDC)── MCU
strategy_manager ◀──vision_result─────────────── vision_node
```
