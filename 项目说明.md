# 项目说明

## 项目概述

四足机器狗（带机械臂与气泵）上位机 ROS2 工作空间。下位机已完成四腿运动学/步态控制、机械臂动作执行、气泵开关，上下位机通过 USB CDC 虚拟串口全双工通信。

**上位机职责**：视觉识别（YOLO 目标检测/3D定位）、坐标系转换、串口通信桥接、高层路径规划与策略执行。

## 技术栈

- **ROS2**: Humble / Iron / Jazzy
- **C++17**: 策略控制节点（quadruped_control/strategy_manager）
- **Python 3**: 视觉检测节点 + 串口桥 + Launch 脚本
- **串口通信**: Linux termios（arm_serial_bridge），Windows 下开发用 pytest
- **构建工具**: colcon (--symlink-install)
- **3D 目标检测**: YOLO + 点云定位（detection_3d）
- **待集成**: Nav2, BehaviorTree.CPP

## 协议

上下位机统一走 `0x55 0xAA` 帧格式，定义在 [protocol.py](src/detection_3d/detection_3d/protocol.py)。

### Host → MCU

| func_id | 名称 | payload |
|---------|------|---------|
| `0x12` | 机械臂目标坐标(相机坐标系) | `3×f32 (x,y,z)` |

### MCU → Host

| func_id | 名称 | payload |
|---------|------|---------|
| `0x21` | 抓放标志 | `u8 flag`  flag=0 抓取/1 放置 |

## 运行方式

### 编译

```bash
cd /path/to/gsing_dog
colcon build --symlink-install
source install/setup.bash
```

### 启动

```bash
# 策略节点
ros2 launch quadruped_bringup robot.launch.py

# 视觉检测 + 串口桥
ros2 launch detection_3d bringup.launch.py use_arm_bridge:=true
```

### Topic 拓扑

```
yolo_detector ──Detection2D──▶ detection_3d_calculator ──Detection3D──▶ arm_serial_bridge ──0x12──▶ MCU
                                                                        arm_serial_bridge ◀──0x21──  MCU
```
